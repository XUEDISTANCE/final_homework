# final_homework
小组大作业：人体识别与跟踪

该存储库包含使用 YOLO-NAS 对象检测模型和 DeepSORT 算法在视频中进行对象跟踪的代码。该代码处理视频的每一帧，使用 YOLO-NAS 执行对象检测，并使用 DeepSort 跨帧跟踪检测到的对象。它还在对象上绘制边界框和标签，并计算处理视频的每秒帧数 (FPS)。
